Start lumen server with local ip:
php -S localhost:8000 -t public  

Clear cache :
php artisan cache:clear 

Use Eloquent ORM in cli (SUPER USEFUL!!!!):
php artisan tinker

You can find all artisan commands by typing this : 
php artisan list
